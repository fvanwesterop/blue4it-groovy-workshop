== 3. Groovy Programer in 4 Seconds


=== 3.1 TrailCalendar
Ga naar de root van je workshop directory voor `lab01`. We maken een simpele evenementen calender met cross-lopen, genaamd Trail Calendar.
[[app-listing]]
[source,bash]
----
$ cd <workshop_home>

$ git checkout lab-01
$ cd workspace/lab-01/trailcalendar
$ gradle init
----
Vervang de inhoud van de file `build.gradle` met de volgende inhoud:
[[app-listing]]
[source,groovy]
.file: 'build.gradle'
----
include::{sourcedir}/labs/lab-01/build.gradle[]
----
